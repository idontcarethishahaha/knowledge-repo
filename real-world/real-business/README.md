# 真实业务

记录实际项目中的业务逻辑、踩坑记录，实战经验积累。

## 记录内容

| 类型 | 说明 |
|------|------|
| 业务逻辑 | 复杂业务流程梳理、状态机设计、权限模型等 |
| 踩坑记录 | 线上问题排查、第三方 SDK 坑点、浏览器兼容 |
| 重构记录 | 代码重构前后对比、为什么这样改 |
| 性能调优 | 慢接口优化、SQL 优化、首屏优化具体案例 |

## 与 case-studies 的区别

| 目录 | 侧重点 |
|------|--------|
| `case-studies/` | **项目级**复盘：从 0 到 1、架构演进、技术选型 |
| `real-business/` | **代码级**记录：具体业务模块、bug 修复、边界处理 |
