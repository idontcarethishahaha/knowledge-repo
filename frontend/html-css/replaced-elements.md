## 置换元素

置换元素是指其内容不由 CSS 控制、而是 **由外部资源（或浏览器内置规则）决定的 HTML 元素**。你可以把它理解成 “**一个被外部内容替换掉的容器**”—— 元素本身只是一个 “占位符”，**实际显示的内容来自外部**（比如图片文件、表单控件的默认样式）。

常见的置换元素
- 媒体类：`<img>`（内容是图片文件）、`<video>`、`<audio>`、`<iframe>`
- 表单类：`<input>`（文本框、按钮、复选框等）、`<select>`、`<textarea>`、`<button>`
- 其他：`<object>`、`<embed>`、`<canvas>`（部分场景下）
