## 经典弹性三栏布局

基于 Flexbox（弹性布局）实现的经典三栏布局—— 也就是「左侧宽度固定、右侧宽度固定、中间区域自适应填充剩余空间」的布局方式，这是前端开发中最常用、最易维护的弹性三栏布局方案

- 给父容器开启 display: flex（弹性布局）
- 让三个子元素横向排列：
- 左、右子元素设置固定宽度（比如 200px）；
- 中间子元素设置 flex: 1，自动占满父容器的剩余空间（这是弹性布局的核心）。


> 这种方案相比传统的「浮动 + BFC」「定位」方案，优势是：代码极简、天然等高、适配性强、无需处理塌陷 / 重叠问题。

<br/>

1. **核心原理：flex: 1 的作用**
中间的 .center 设置 flex: 1（等价于 flex: 1 1 0%），意味着：
父容器的剩余宽度 = 容器总宽度 - 左侧宽度 - 右侧宽度 - gap 间距；
这些剩余宽度会全部分配给中间区域，无论窗口放大 / 缩小，中间都会自适应。

<br/>

2. **天然等高特性**
**Flex 布局下，子元素默认会继承父容器的高度**（align-items: stretch 是默认值），所以左、中、右三栏会自动等高 —— 这是传统布局（浮动 / BFC）很难做到的，也是 Flex 布局的一大优势。
