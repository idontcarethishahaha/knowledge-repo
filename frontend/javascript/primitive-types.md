## 基本数据类型

JS 的数据类型分为两大类：**基本数据类型** 和 **引用数据类型**（对象类型），其中基本数据类型是基础，共 **7** 种：

|基本数据类型|	String	|字符串（如 "hello"、'123'）|
|:--:|:--:|:--:|
|               |Number|	数值（整数 / 浮点数 / 特殊值，如 10、3.14、NaN、Infinity）|
|               |Boolean|	布尔值（true /false）|
|               |Undefined|	未定义（变量声明但未赋值时的默认值）|
|               |Null	|空值（表示 “无”，手动赋值的空引用）|
|               |Symbol（ES6+）|	唯一标识（即使值相同，也不相等，如 Symbol ('a') !== Symbol ('a')）|
|               |BigInt（ES10+）|	大整数（解决 Number 最大安全值限制，如 9007199254740991n）|
|引用数据类型|	Object|	包含普通对象、数组（Array）、函数（Function）、日期（Date）、正则（RegExp）等|

### 基本数据类型特性

##### 1. 存储方式：栈存储（Stack）
基本数据类型的值直接存储在栈内存中，特点：
- 占用空间小、大小固定；
- 访问速度快；
- 赋值时是「值拷贝」（修改新变量不会影响原变量）

```js
let a = 10;
let b = a; // 把a的值（10）拷贝给b
b = 20;    // 修改b，a不受影响
console.log(a); // 10
console.log(b); // 20
```

<br/>

##### 2. 类型判断特殊：typeof null === 'object'（JS 历史 bug）


|特性|	基本数据类型|	引用数据类型|
|:--:|:--:|:--:|
|存储位置|	栈内存|	堆内存（栈存引用地址）|
|赋值方式|	值拷贝（独立）	|引用拷贝（共享同一内存）|
|比较方式|	比较值是否相等|	比较引用地址是否相等|
|可变性|	不可变（修改是重新赋值）|	可变（直接修改堆内存数据）|

- **基本类型不可变**
```js
let str = 'hello';
str.toUpperCase(); // 不会修改原字符串
console.log(str); // 'hello'
str = str.toUpperCase(); // 重新赋值，才会改变
console.log(str); // 'HELLO'
```
